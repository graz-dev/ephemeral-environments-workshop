What is a Composition?

Composition in Crossplane is a resource that defines how to implement an abstract API (XRD) using real infrastructure resources.

  • Infrastructure Mapping: Connects your custom API to one or more managed resources (e.g. databases, clusters).
  • Reusable Blueprints: Platform teams can offer standardized infrastructure patterns to developers.
  • Patching & Transformations: Allows mapping, patching, and transforming fields from the abstract API to the underlying resources.
  • Multi-resource: A single Composition can create and manage multiple resources for each custom object.
